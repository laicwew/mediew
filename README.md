# Mediew

具有瀑布流和网格风格，简洁轻量的图片/视频浏览器。

## 功能特性

- **文件夹导航** - 左侧树形目录，单击预览
- **图片预览** - 支持 JPG、PNG、GIF、WebP、BMP 格式
- **视频播放** - 支持 MP4、WebM、MOV、AVI、MKV、WMV 格式(大概吧)
- **主题切换** - 深色/浅色主题（Twitter/X风格）
- **布局模式** - 瀑布流混合布局、网格布局、文件名模式
- **排序分组** - 按修改时间或文件名排序，支持年/月/日/时分组
- **图片操作** - 缩放、拖拽、双击全屏
- **视频播放器** - 进度条拖拽、音量控制、速率控制、全屏播放、画中画播放
- **文件操作** - 重命名、删除、拖拽移动到左侧目标文件夹，，批量拖拽移动和删除
- **文件夹操作** - 在文件树中新建、重命名、删除文件夹，拖拽文件夹移动，批量拖拽移动和删除，打开文件资源管理器
- **自动刷新** - 文件夹内容变化自动更新

## 操作说明

| 操作 | 效果 |
|------|------|
| 单击文件夹 | 查看该文件夹内的图片 |
| 双击图片/视频 | 打开预览 |
| 预览界面-双击图片 | 缩放切换 |
| 预览界面-双击视频 | 全屏播放 |
| 左右方向键 | 上/下一张图片，或视频快进/后退5秒 |
| 空格键 | 播放/暂停视频 |
| ESC | 退出全屏或关闭预览 |
| 右键菜单 | 重命名（仅限按名称排序模式）、删除 |
| 右键文件夹 | 新建文件夹、重命名、删除文件夹、打开文件资源管理器 |
| 拖拽文件夹 | 移动文件夹到目标位置 |
| Ctrl+滚轮 | 缩放主面板视图大小 |

## 安装运行

```bash
# 安装依赖
npm install

# 启动应用
npm start
```
